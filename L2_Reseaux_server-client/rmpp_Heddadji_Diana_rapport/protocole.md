<p>Proposition de protocole pour la communication inter-serveurs/clients pour le projet de chat textuel.</p>
<p>Quelques règles pour assurer le bon fonctionnement du protocole :</p>
<ul>
<li>Chaque ligne se termine par les caractères <code>\r\n</code>.</li>

<li>Un nom d’utilisateur contient de 1 (un) à 20 (vingt) caractères.</li>
<li>Un message contient de 1 (un) à 2000 (deux mille) caractères (hors protocole)</li>
</ul>
